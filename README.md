
# Simplified-Resume-CV-Pages-portfolio

A sleek, dark-themed professional portfolio with built-in resume/CV upload and download functionality. Share your portfolio link with recruiters - they can view your profile and download your documents instantly.

##  Features

- **🎨 Stunning Dark Theme** - Green accents with glowing animations
- **📄 Resume/CV Hosting** - Upload once, share forever
- **🔒 Secure Storage** - Files stored locally in browser (IndexedDB)
- **📱 Fully Responsive** - Works on all devices
- **⚡ Zero Backend** - No server, no database, no dependencies

##  Quick Start

### Option 1: Direct Open

# Clone or download
git clone https://github.com/yourusername/simon-portfolio.git
cd simon-portfolio

# Open directly
open index.html


### Option 2: Local Server

# Python
python -m http.server 8000

# Node.js
npx serve .

# VS Code Live Server
# Right-click index.html → "Open with Live Server"

## ⚙️ Configuration

Edit `config.js` to customize:
- Professional details (name, title, contact)
- Social media links
- File upload limits
- Feature flags

## 🔒 Privacy & Security

- **100% Client-Side** - No data leaves your browser
- **IndexedDB Storage** - Files stored locally (up to 50MB)
- **No Tracking** - Zero analytics or third-party scripts
- **User Control** - Delete files anytime with one click

## 📤 How to Use

### For You (Portfolio Owner)
1. Open portfolio in browser
2. Navigate to **Resume** or **CV** page
3. Click **Choose File** and upload your document
4. Share your portfolio URL with recruiters

### For Recruiters
1. Visit your portfolio URL
2. Browse your profile and skills
3. Navigate to **Resume** or **CV** page
4. Click **Download** to get your document
